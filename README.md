# 🚀 Interview Preparation Agent

AI-powered interview coach that generates role- and company-specific questions, simulates mock interviews, and evaluates answers with scoring and feedback.

## 🔧 Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI API

## ▶️ How to Run
pip install -r requirements.txt
streamlit run app.py
